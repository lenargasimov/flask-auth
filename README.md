# Installation on Linux and Unix like OS

```
git clone https://github.com/lenargasimov/flask-auth.git
cd flask-auth
pip3 install -r requirements.txt 
python3 main.py
```

# Installation on Windows

```
git clone https://github.com/lenargasimov/flask-auth.git
cd flask-auth
pip install -r requirements.txt 
python main.py
```
